### Colocar estilos en el html directamente
Se puede colocar un cierto estilo a una etiqueta html usando el atributo 'style':
`<div style="background-color: red;"></div>`

### Colocar hoja de estilos incrustada en el html
```
<head>
    <link...>
    <style>
        ...
    </style>
</head>
```
